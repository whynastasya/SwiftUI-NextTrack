# «Следующий Трек»

Повторяем анимацию кнопки следующего трека из Apple Music.

- Структура Button. Нельзя использовать жесты.
- У иконки есть инерциальный отскок в конце анимации.
- Опционально: повторить круглый фон при нажатии.
